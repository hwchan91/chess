# Chess

A game of chess for 2 persons played with algebriac notation([see Wikipedia: Algebriac Notation](https://en.wikipedia.org/wiki/Algebraic_notation_(chess))).

Working example: https://repl.it/HpML/6

## Usage

```
ruby chess.rb
```
